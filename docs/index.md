# Qiskit QM Provider — Documentation

This site contains the **API documentation** for the [qiskit-qm-provider](https://github.com/arthurostrauss/qiskit-qm-provider) package: a bridge between the Qiskit ecosystem and Quantum Machines' Quantum Orchestration Platform (QOP).

## Contents

- **[Providers](providers.md)** — `QMProvider`, `QmSaasProvider`, and `IQCCProvider`
- **[Backend](backend.md)** — `QMBackend`, utilities (`add_basic_macros`, `get_measurement_outcomes`), and `QMInstructionProperties`
- **[Primitives](primitives.md)** — `QMEstimatorV2` and `QMSamplerV2`
- **[Parameter Table](parameter_table.md)** — `ParameterTable` and `Parameter` for real-time parameters

For installation, quick start, and high-level usage, see the main [README](https://github.com/arthurostrauss/qiskit-qm-provider#readme) in the repository.
